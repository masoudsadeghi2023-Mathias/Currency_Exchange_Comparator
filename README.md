# Currency Exchange Comparator 💱

A simple and clean Streamlit app for comparing currency exchange values between:
- SEK → EUR (EX)
- SEK → Toman (EX1)
- EUR → Toman (EX2)

The app supports:
- Multilingual interface (EN / FA)
- Light/Dark theme toggle
- Clean UI with formatted results
- Mobile-friendly layout

---

## 🚀 Run locally

```bash
pip install streamlit
streamlit run app.py
