# Currency Exchange Comparator 💱

A simple and clean Streamlit app for comparing currency exchange values between:
- SEK → EUR (EX)
- SEK → Toman (EX1)
- EUR → Toman (EX2)

The app supports:
- Multilingual interface (EN / FA)
- Light/Dark theme toggle
- Clean UI with formatted results
- Mobile-friendly layout

---

## 🚀 Run locally

```bash
pip install streamlit
streamlit run app.py

# 💱 مقایسه‌گر نرخ تبدیل ارز (Currency Exchange Comparator)

یک اپلیکیشن ساده، زیبا و سریع با Streamlit برای مقایسه نرخ تبدیل بین:
- کرون سوئد → یورو (EX)
- کرون سوئد → تومان (EX1)
- یورو → تومان (EX2)

این برنامه برای افرادی طراحی شده که می‌خواهند سریع و دقیق اختلاف نرخ‌ها را بررسی کنند.
